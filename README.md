## ✦ Sobre Nós

**Lumina Studio Ltda** é uma empresa de desenvolvimento de software fundada por 5 sócios apaixonados por tecnologia e inovação. Atuamos em duas frentes de mercado — **desenvolvimento web de alto padrão** e **criação de experiências imersivas no Roblox** — entregando soluções que combinam excelência técnica com design de impacto.

> Nossa missão: **iluminar cada projeto com criatividade, precisão e resultado.**
